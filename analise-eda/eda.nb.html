Aqui temos alguns dados gerais como a mdia e mediana de respostas por reclamao, assim como a quantidade de respostas por reclamao. Esse dado utilizado para saber se a amostra que pegamos representativa o suficiente para podermos afirmar que h um consenso em relao insatisfao de uma reclamao

avaliacoes %>% 
    group_by(id_reclamacao) %>% 
    count() %>% 
    ggplot(aes("reclamacoes", n)) + 
    geom_jitter(width = .05, alpha = .7)

avaliacoes %>% 
    group_by(id_reclamacao) %>% 
    count() %>%
    ungroup() %>% 
    summarise(media = mean(n), 
              mediana = median(n))

A partir de tais dados, possvel dizer que houve um consenso no sentido de que existe um consenso se a diferena entre o terceiro quartil e primeiro quartil seja < 1. Para analisar os lxicos iremos normalizar o resultado dado pelos mesmos para que eles se adequem nossa escala. Em seguida, consideraremos um acerto se o resultado normalizado estiver dentro do primeiro e terceiro quartil.

avaliacoes %>% 
    ggplot(aes(x = reorder(id_reclamacao, insatisfacao, fun = median), y = insatisfacao)) +
